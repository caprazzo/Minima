About Minima

  • Add with one click
  • Drag to organize
  • Archive when done
  • Show archived stuff if you want

Flexible

  • Create as many lists or notes you like
  • The layout adapts to the browser size

Always fresh

  • When another user makes a change, it is immediately displayed on your board

Install

Minima comes as a standlone server with an embedded database. Get up and running in seconds.

  • Download the latest build
  • java -jar Minima-0.9-standalone.jar (or doubleclick on the jar)
  • Browse to http://localhost:8989/index
  • Bravo!

Options

Options are passed as Jvm arguments.

For example, to start the server on port 8080, use java -jar -Dminima.port=8080 Minima-standalone.jar

minima.port
http port, defaults to 8989
minima.db.dir
database directory, defaults to ./minima-db and is created if it does not exist
minima.password
By default Minima allows anyone to see the board and make changes. If a password is set, it will be requested to see and modify the board.
minima.readonly
if this is set to true and a password is configured, non-authenticated users can see but not modify the board
